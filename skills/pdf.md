# Skill Card: pdf

Purpose: read, create, review, and visually inspect PDF files where layout matters.

Best for:

- PDF text and table extraction.
- Page rendering and screenshot review.
- Generated PDF layout verification.
- Visual checks for figures, typography, and pagination.

Typical tools:

- PyMuPDF
- pdfplumber
- pypdf
- reportlab
- Poppler rendering tools

Quality rules:

- Prefer rendered-page visual checks when layout matters.
- Keep extraction and rendering artifacts separate from private source PDFs when publishing.
